Running ROS Noetic on an RK3328 Board

I was recently set the challenge of installing ROS Noetic onto a RK3328 board. I can’t yet reveal the product specifically, but I can document some of how I installed it. The board itself runs Debian with a 4.4.194-1-rockchip aarch64 kernel.

You can effectively follow the ROS instructions from the ROS wiki to get everything installed. There are some pre-reqs that I found and those are included in the full list below.

How does it work? Great! It runs ROS without issue. There are some oddities with the board itself and peripherals that need to be resolved but those will happen in time. It’s a very nice little setup and I can recommend it.
